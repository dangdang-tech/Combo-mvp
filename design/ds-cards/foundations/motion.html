Motion

--cb-duration-fast / base / slow × --cb-ease-standard / decelerate

滑块单程移动时长严格等于 duration token（120 / 200 / 320ms），停留段仅为观察留白，整周期为 token × 4。动画的 duration 与 timing-function 全部引用 var(--cb-duration-*) / var(--cb-ease-*)。

--cb-ease-standard
cubic-bezier(0.2, 0, 0, 1)
--cb-duration-fast · 120ms
--cb-duration-base · 200ms
--cb-duration-slow · 320ms
--cb-ease-decelerate
cubic-bezier(0, 0, 0.2, 1)
--cb-duration-fast · 120ms
--cb-duration-base · 200ms
--cb-duration-slow · 320ms
